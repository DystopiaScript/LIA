## Estructura de la Matriz de Transición

```
Estado Actual × Carácter de Entrada → Siguiente Estado
```

## Verificación de la Matriz

Para cada estado, verificar:
1. ✅ Todos los caracteres posibles tienen transición definida
2. ✅ Estados de aceptación (100-199) están correctamente mapeados
3. ✅ Estados de error (500+) manejan casos inválidos
4. ✅ Estado 0 es el estado inicial
5. ✅ Loops están correctamente implementados (ej: estado 1 → estado 1 con L|l|d)
